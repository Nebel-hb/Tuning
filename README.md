# Tuning

## サイト概要
- アマチュア音楽家たちのコミュニティサイト

### サイトテーマ
- 共に演奏する仲間を探す

### テーマを選んだ理由
- 自分が所属しているオーケストラにて演奏会を開催するにあたって、毎年足りない楽器のメンバーを探すことに苦労していることから、音楽家の繋がれるアプリケーションを作成することにしました。

### ターゲットユーザ
- 共演できる相手を探しているアマチュア音楽家

### 主な利用シーン
- オーケストラ、アンサンブル、伴奏者等、編成メンバーを探す
